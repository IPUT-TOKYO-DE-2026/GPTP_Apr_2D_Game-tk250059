# ESCAPE THE TRIANGLE

## ゲームの内容
プレイヤーは広大なマップに落ちている3つのカギを探して、ゴールを目指すゲーム。3つ集めなくちゃゴールはできない。マップにはプレイヤーの脅威となる敵がいる。捕まったらその場でゲームオーバー。

## 操作説明
- WASDでの前後左右移動
- 黒は障害物で通れない
- 赤い場所はハイド場所。敵の追跡から逃れられる
